# KiddoLearn – Interactive Quiz Learning App

KiddoLearn is a cross-platform educational quiz application designed for children to learn in a fun, interactive way.  
Built using React Native (frontend) and Node.js + Express + MongoDB (backend), the app provides multiple quiz categories, dynamic scoring, sound effects, and a smooth mobile learning experience on both Android and iOS.

## Features

- **Multiple Quiz Categories**  
  Science, Math, Grammar, Aptitude, and Image-based quizzes.

- **Level-Based Learning**  
  Basic, Medium, and Hard levels for selected quizzes.

- **User Authentication**  
  Secure login system using email & password.

- **Dynamic Question Handling**  
  Auto-scoring logic with instant result display.

- **Interactive Sound Controls**  
  Users can play or stop background audio during quizzes.

- **Reset / Previous / Next Navigation**  
  Smooth question-to-question transitions.

- **Cloud-Ready Backend**  
  Quiz data, user data & feedback stored in MongoDB Atlas.

- **Feedback Form**  
  Users can submit comments about their experience.

---

## Technologies Used

- **Frontend:**  React Native, JavaScript, React Navigation, AsyncStorage    
- **Backend:** Node.js, Express.js, MongoDB, REST API Architecture  
- **Other:** dotenv for environment variables, CORS
